# pong

To run this game, you must have PyGame and PyGame Menu installed.\
[PyGame Install](https://www.pygame.org/wiki/GettingStarted)\
[PyGame Menu Install](https://pypi.org/project/pygame-menu/)

## Controls
Left Paddle:\
    * Move Up = W\
    * Move Down = S\
Right Paddle:\
    * Move Up = UP ARROW\
    * Move Down = DOWN ARROW

Enjoy!
